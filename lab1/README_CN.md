# 机器学习基础实验说明文档

## 项目概述

本项目是一个机器学习入门实验（Lab1），主要涵盖数据加载、可视化、分类模型和过拟合概念。

## 项目内容

### 1. 加载和可视化Iris数据集

- 使用sklearn内置的Iris数据集
- 构建pairplot（配对图）进行数据探索性分析
- Iris数据集包含150个样本，4个特征（花萼长度、花萼宽度、花瓣长度、花瓣宽度），3个类别（setosa、versicolor、virginica）

### 2. 逻辑回归二分类

- 选择Versicolor和Virginica两类进行二分类
- 使用train_test_split将数据分为训练集（70%）和测试集（30%）
- 训练逻辑回归模型并评估准确率
- 绘制混淆矩阵可视化分类结果

### 3. 过拟合演示

- 生成合成数据：y = 2x + 1 + 噪声
- 使用不同次数的多项式特征进行线性回归：
  - 次数=1：欠拟合
  - 次数=3-10：良好泛化
  - 次数=15：过拟合
- 通过R²分数对比展示过拟合现象

## 理论要点

1. **过拟合问题**：模型过于复杂，记住了训练数据中的噪声，导致在新数据上表现差
2. **为何不能用训练数据评估模型**：因为模型已经"见过"这些数据，评估结果会过于乐观
3. **正则化**：通过惩罚模型复杂度来防止过拟合
